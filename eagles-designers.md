# Обзор

## Создание кампании

расположена на странице https://ads.vk.com/hq/overview

![alt text](image.png)

### Настройка кампании

2. Нажатие на кнопку `Создать компанию` внутри контейнера `Кампании` открывает страницу создания компании
![Alt text](image-1.png)

3. Нажатие внутри раздела `Целевые действия` на кнопку `Лид-формы и опросы` раскрывает контейнер `Объект рекламы`
![Alt text](image-3.png)

4. Нажатие на раскрывающийся список `Лид-форма` внутри `Объект рекламы` открывает список
![Alt text](image-4.png)

5. Нажатие на `Создать лид-форму` открывает модальное окно `Новая лид-форма`
![Alt text](image-5.png)

#### Вопросы

1. Нажатие кнопки `Продолжить` внутри `Оформление` открывает раздел `Вопросы`
![Alt text](image-12.png)

2. Нажатие кнопки `Добавить вопрос` добавляет контейнер `Вопрос`
![Alt text](image-13.png)
3. Ввод текста в поля `Напишите вопрос` и `Введите ответ` в реальном времени изменяет соответствующие элементы внутри макета
![Alt text](image-14.png)
5. Нажатие на кнопку `Добавить ответ` добавляет поле для еще одного вопроса
![Alt text](image-15.png)
6. Нажатие на кнопку удаления ответа - удаляет ответ

![Alt text](image-16.png)

7. Нажатие на кнопку `Добавить контактные данные` открывает модальное окно с выбором информации
![Alt text](image-17.png)
8. Выбор `электронная почта` и нажатие `Добавить` добавляет новый элемент в `Контактная информация`
![Alt text](image-18.png) 
9. Нажатие на кнопку удаления контактной информации - удаляет элемент
![Alt text](image-19.png)

#### Результат

1. Нажатие кнопки `Продолжить` внутри `Вопросы` открывает раздел `Результат`
![Alt text](image-20.png)

2. Нажатие на кнопки `Добавить ссылку на сайт`, `Добавить телефон` и `Добавить промокод` заменяет кнопки на соответствующие поля ввода
![Alt text](image-21.png)

#### Настройки

1. Нажатие кнопки `Продолжить` внутри `Результат` открывает раздел `Настройки`
![Alt text](image-27.png)

2. Нажатие на чекбокс `Уведомлять о новых заявках по email` добавляет поле ввода почты
![Alt text](image-28.png)
3. Нажатие на чекбокс `Обязательные вопросы` открывает модальное окно подтверждения выбора
![Alt text](image-29.png)

4. Нажатие кнопки `Сохранить` закрывает модальное окно создания лид-формы и добавляет ёё
![Alt text](image-30.png)

### Группы объявлений 

1. Нажатие на кнопку `Продолжить` в разделе `Настройка кампании` открывает раздел `Группы объявлений`
![Alt text](image-33.png)

2. Ввод в поле `Регионы показа` отображает подходящие варианты в выпадающем списке, нажатие на вариант добавляет его как выбран
![Alt text](image-34.png)

### Завершение создания кампании

1. Нажатие кнопки `Опубликовать` создает кампанию и переадресует на https://ads.vk.com/hq/overview
![Alt text](image-48.png)

### Удаление кампании

1. Нажатие кнопки `Не транслируются` в контейнере `Кампании` открывает страницу с созданными кампаниями
![Alt text](image-49.png)

2. Нажатие на чекбокс по кампании и выбор в качестве действия `удалить` - удаляет кампанию
![Alt text](image-50.png)
