## 5. Создание объявления

## Чек-лист: Создание объявления

### 1. Этап: Начало создания кампании

- Перейти в раздел "Кампании" и нажать "Создать кампанию"
  - Открывается форма выбора цели
- Выбрать цель "Целевые действия" и нажать "Продолжить"
  - Статус этапа "Группы объявлений" подсвечивается синим кружком
- На экране выбора рекламного объекта нажать карточку "Сайт"
  - Форма заполняется, ранее введенные значения сохраняются

### 2. Этап: Настройка группы объявлений

- Ввести название группы менее 3 символов
  - Появляется сообщение об ошибке, сохранение недоступно
- Ввести валидное название (например, "Моя группа") и кликнуть вне поля
  - Название сохраняется автоматически
- Развернуть раздел "География показа"
  - Отображаются поля выбора страны, региона и города
- Ввести "Чувашская Республика" в поиске
  - Выбранный регион отображается в списке выбранных
- Открыть раздел "Интересы и поведение"
  - Появляются категории: "Интересы", "Ключевые фразы", "Сообщества", "Музыканты"
- В разделе "Интересы" ввести "Мебель" и выбрать "Мебель и интерьер"
  - Категория добавляется, аудиторный охват обновляется
- Развернуть раздел "Устройства"
  - Отображаются чекбоксы: "Мобильные", "Десктопные"
- Развернуть раздел "Места размещения", отключить автоселект
  - Доступны варианты: ВКонтакте, Одноклассники, Проекты VK, РСЯ
- Нажать "Продолжить"
  - Этап "Объявления" становится активным и подсвечен синим

### 3. Этап: Настройка объявления

- Загрузить логотип в формате JPG размером больше 100x100 px
  - Превью отображает загруженный логотип
- Ввести заголовок "Стулья"
  - Текст проходит валидацию, отображается на превью
- Ввести описание "Продаём стулья из красного дерева"
  - Текст отображается на карточке объявления
- Проверить поле "Ссылка на сайт"
  - Адрес, указанный на предыдущем шаге, подставляется автоматически
- Выбрать надпись на кнопке "Заказать"
  - Превью кнопки изменяется согласно выбору
- Загрузить изображение в раздел "Медиафайлы"
  - Предлагаются варианты обрезки и анимации
- Выбрать пятый слайд в карусели
  - Превью показывает соответствующее изображение
- Нажать "Добавить ещё формат", выбрать "Галерея", загрузить 2 изображения
  - В режиме "Слайдер" отображаются загруженные изображения

### 4. Финализация и ИНН

- Нажать "Опубликовать"
  - Появляется окно с вводом ИНН
- Ввести ИНН с длиной менее 12 символов (например, "123")
  - Появляется ошибка: "Длина ИНН должна быть 12 символов"
- Ввести в ИНН буквы или символы (например, "$2@-")
  - Появляется ошибка: "Некорректный ИНН"
- Закрыть окно ИНН без сохранения
  - Все данные объявления остаются без изменений
- Нажать "Сохранить черновик"
  - Появляется уведомление "Изменения сохранены"

### 5. UX и общие проверки

- Проверить, что каждый этап создания отображается в статусной строке
  - Пройденные этапы отмечаются галочками, текущий — синим кружком
- Вернуться к предыдущему этапу и снова открыть текущий
  - Все ранее введённые данные подгружаются корректно
- Проверить поведение при обновлении страницы на каждом этапе
  - Данные не теряются, форма восстанавливается

## Чек-лист: Создание аудитории

### Общие проверки

- Переход в раздел "Аудитории"
  - Отображается заглушка "Аудиторий пока нет" с кнопкой "Создать аудиторию", справочной ссылкой и меню дополнительных действий
- Нажать "Создать аудиторию"
  - Открывается форма с полем ввода названия, кнопками добавления/исключения источников и действиями "Сохранить" / "Отмена"
- Ввести уникальное название (до 255 символов)
  - Ввод сохраняется без ошибок, счетчик символов отображается
- Оставить поле названия пустым
  - Кнопка "Сохранить" отключена или появляется предупреждение
- Ввести название длиннее 255 символов
  - Поле выделяется, появляется ошибка: "Максимальная длина 255 символов"
- Нажать "Сохранить" без выбора источников
  - Отображается сообщение об обязательности добавления хотя бы одного источника
- Нажать "Отмена"
  - Возврат на предыдущий экран без создания аудитории
- Попытка закрыть окно при внесенных изменениях
  - Появляется диалог подтверждения отмены изменений

### Работа с источниками

- Нажать "Добавить источник"
  - Открывается меню с категориями источников, такими как "Списки", "Поведение", "Интересы"
- Нажать "Исключить источник"
  - Аналогично открывается меню исключения источников
- Добавить источник и исключить тот же источник
  - Поля выделяются, появляется ошибка о невозможности дублирования
- Выбор источника "Категории мобильных приложений"
  - Заполняются обязательные поля, сохраняется источник
- Удалить добавленный источник
  - Появляется окно подтверждения удаления источника

### Проверки с различными источниками

- Добавить пользователей из сообщества вручную
  - Источник сохраняется, возвращение к форме создания аудитории
- Загрузить новый список пользователей (CSV или TXT)
  - Происходит загрузка, валидные строки обрабатываются, ошибки выводятся
- Добавить источник "События в рекламной кампании"
  - Источник сохраняется при корректных данных
- Добавить источник "Музыканты"
  - Успешное добавление исполнителей и возврат к созданию
- Исключить пользователей, открывших лид-форму
  - Источник применяется, данные отображаются

### Негативные сценарии

- Загрузка пустого файла
  - Отображается сообщение об ошибке: "Файл пуст"
- Ввод ID сообщества с ошибкой в формате
  - Источник не добавляется, появляется предупреждение
- Источник требует настройки (например, пиксель не подключен)
  - Выдается сообщение о необходимости предварительной настройки

### Действия после создания

- Аудитория отображается в списке
  - Название, тип, охват, ID и дата создания отображаются корректно
- Использование поиска по названию
  - Результаты фильтруются, ошибки при длинных строках отображаются корректно
- Удаление аудитории через меню
  - Появляется окно подтверждения, после удаления элемент исчезает
- Использование массовых действий (удалить/поделиться)
  - Подсказки появляются при наведении, действия выполняются только при выборе аудитории

### Доступ и совместная работа

- Настройка доступа к аудитории
  - Появляется меню с выбором ID других кабинетов
- Попытка сохранить доступ без заполнения всех данных
  - Поля подсвечиваются, отображается сообщение "Нужно заполнить"
