###### Оформление
- [ ] Окно создание лид-формы закрывается при нажатии на кнопку "Отмена"
- [ ] Ошибкаи в полях "Логотип", "Название компании (организации)", "Заголовок" и "Описание формы" при нажатии "Продолжить" если поля пустые

- [ ] Ошибка "Нужно заполнить" рядом с полем выбора логотипа исчезает при выборе логотипа
- [ ] Происходит переход в раздел "Вопросы" при нажатии на кнопу "Продолжить" если нужные поля заполнены
- [ ] Счетчик количества введенных символов совпадает с их действительным количеством при введении текста в поля
- [ ] Ошибка при нажатии на кнопку "Продолжить" с превышенным пределом введенных символов в полях

###### Вопросы
- [ ] Происходит переход в раздел "Оформление" при нажатии на кнопку "Назад"
- [ ] Появляется форма создания вопроса с заголовком "Вопрос №1" при нажатии на кнопку "Добавить вопрос"  
  ###### Форма создания вопроса
  - [ ] Ошибка если форма пустая
  - [ ] Переход в раздел "Результат" при нажатии на кнопку "Продолжить" если все формы создания вопросов заполнены

###### Результат
- [ ] Поле "Заголовок" заполнено текстом "Спасибо за ответы!" при первом переходе в раздел "Результат"
- [ ] Поле "Описание" заполнено текстом "Заявка отправлена" при первом переходе в раздел "Результат"
- [ ] Происходит переход в раздел "Настройки" при нажатии на кнопку "Продолжить"

###### Настройки
- [ ] Ошибка при нажатии на кнопку "Сохранить" если обязательные поля пустые
- [ ] Окно создания лид-формы закрывается при нажатии на кнопку "Сохранить" если обязательные поля заполнены

- [ ] Лид-форма появляется в списке после создания лид-формы

***Должна быть создана лид-форма***

##### Редактироваоть лид-форму
  ###### Оформление

  - [ ] Название лид-формы в списке изменяется после изменения поля "Название лид-формы" и сохранения изменений

- [ ] Появляется модальное окно подтверждения сохранения при нажатии на кнопку закрытия окна редактирования лид-формы если поля были изменены
  ###### Модальное окно сохранения лид-формы
  - [ ] Модальное окно закрывается при нажатии на кнопку закрытия
  - [ ] Изменения сохраняются при нажатии на кнопку "Сохранить лид-форму"
